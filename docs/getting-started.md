# EchoMind 上手指南

5 分钟从零到第一条灵感落库。

## 1. 安装

### Windows

1. 从 [Releases](https://github.com/Qc-riviere/EchoMind-public/releases/latest) 下载 `EchoMind_x.y.z_x64-setup.msi`
2. 双击安装
3. **SmartScreen 拦截**？这是因为 v0.2 暂未购买 Microsoft 代码签名证书
   - 点击「**更多信息**」
   - 再点击「**仍要运行**」
   - 安装完成即可正常使用

### macOS

1. 从 [Releases](https://github.com/Qc-riviere/EchoMind-public/releases/latest) 下载 `EchoMind_x.y.z_universal.dmg`
2. 拖入 Applications 文件夹
3. **Gatekeeper 拦截**？这是因为 v0.2 暂未购买 Apple Developer 签名（$99/年）
   - 在 Finder 中找到 EchoMind.app
   - **右键** → 选「**打开**」
   - 弹窗确认 → 再点「**打开**」
   - 之后双击启动即可

> v0.3 Beta 起将完成两个平台的代码签名。

## 2. 首次启动

启动后自动弹出 **4 步引导**：

| 步骤 | 你需要做的 |
|---|---|
| 1. 欢迎 | 点「下一步」|
| 2. 配置 LLM | 选 **DeepSeek**（最低门槛，¥10 充值即可上手）→ 粘贴 API Key → 点「测试连接」→ ✓ |
| 3. 第一条灵感 | 在输入框打字 → Enter |
| 4. 微信桥 teaser | 看简介；之后再配置 |

### 怎么拿 DeepSeek API Key？

1. 访问 [platform.deepseek.com](https://platform.deepseek.com)
2. 注册账号 → 实名 → 充值 ¥10（够用很久）
3. API Keys 页面 → 创建 API Key → 复制
4. 粘贴回 EchoMind 设置即可

> 也可以用 OpenAI / Claude / Gemini 的 Key，在「设置 → LLM」切换。

## 3. 桌面速记

| 操作 | 快捷键 |
|---|---|
| 唤出速记浮窗（任意应用下） | `Ctrl+Shift+I`（Mac: `Cmd+Shift+I`）|
| 保存灵感并关闭浮窗 | `Enter` |
| 取消并关闭浮窗 | `Esc` |
| 换行（写多行） | `Shift+Enter` |

保存后浮窗会自动消失，托盘图标上的「今日新增」徽标 +1。

## 4. 主界面三大区

```
┌─────────────────────────────────────────────────────┐
│ 侧栏        │  最近 5 条             │ 操作区       │
│            │  ───────────           │              │
│  🏠 首页    │  [灵感 A]              │              │
│  💬 对话    │  [灵感 B]              │              │
│  🌐 微信桥  │  [灵感 C]              │              │
│  🕸️  图谱   │                        │              │
│  ⚙️  设置   │  对话最多 5 条          │              │
│            │  ───────────           │              │
│            │  [灵感 X (3 轮对话)]    │              │
│            │  [灵感 Y (2 轮)]       │              │
└─────────────────────────────────────────────────────┘
```

- 主列表分页 9 条/页，按时间倒序
- 卡片右上 ⋯ hover 即变 ☐ 复选框 → 多选 2-20 条触发 AI 总结
- 任一卡片点「对话」→ 进入深挖模式

## 5. 多选 AI 总结 + 导出

1. 在主列表 hover 卡片右上 ⋯ → 自动变 ☐
2. 勾选 **2–20 条**相关灵感
3. 底栏出现「已选 N · AI 总结」按钮 → 点击
4. 弹窗显示 AI 整理出的中心论点 + 3-5 个要点
5. 三个操作：
   - 💾 **保存为新灵感**（自动打 `ai-summary` 标签）
   - 📤 **导出**：MD / DOCX / **PDF**（PDF 走系统打印对话框，选「Microsoft Print to PDF」）
   - 关闭

## 6. 微信桥（可选）

把第二大脑接到手机微信上，发条消息即落库。

### 步骤

1. 侧栏「微信桥」→ 点「**扫码连接**」
2. 用**你自己的微信**扫码（不要给同事扫，bot 绑定个人号）
3. 手机微信会跳「确认登录电脑」→ 点「登录」
4. 桌面端进度条走完 → 显示「微信已连接」

### 可用命令（在微信里发给 bot）

| 命令 | 含义 |
|---|---|
| `直接发文字` | 快速记录想法 |
| `发送图片` | AI 识别并记录 |
| `/list [n]` | 列出最近 n 条 |
| `/search <关键词>` | 语义搜索 |
| `/view <ID>` | 查看详情 |
| `/chat <ID>` | 开始 AI 对话 |
| `/exit` | 退出对话模式 |
| `/help` | 显示帮助 |

### 桌面关机时也想用？

可以开通 **L2 Bridge** 云端服务（Beta 阶段免费试用，正式版 ¥99/年）：
- 微信消息 → 云端节点临时缓存 → 桌面端拉取后清空
- 云端只做中转，不长期存储

详见「设置 → 云桥」。

## 7. 思维图谱

侧栏「图谱」→ 看你所有灵感按语义相关度连成的网络。

- **节点颜色**：按 domain 主题着色（technology / business / personal …）
- **绿色节点**：近 24 小时新建
- **节点大小**：按内容长度
- **拖动 / 缩放**：自由探索
- **点击节点**：右侧弹抽屉查看详情 + 跳对话

## 8. 设置项速览

| Tab | 干什么 |
|---|---|
| LLM | 切换 provider、改 API Key、调温度 |
| Embedding | 高级：单独配置 embedding（默认与 LLM 共用）|
| AI | 开关：自动打标 / 语义关联 / 自动 enrich |
| Skills | 自定义 markdown skills 扩展 agent 能力 |
| Appearance | 主题（亮/暗/跟随系统）、字号、紧凑模式 |
| Data | 导出 / 导入 / 重建向量索引 / 清空数据 |
| About | 版本、链接 |

## 9. 出问题了？

- **AI 调用报错**：「设置 → LLM → 测试连接」检查 Key
- **微信桥扫不上**：「设置 → 桥接 → 重新绑定」
- **搜索没结果**：「设置 → Data → 重建向量索引」（修维度切换问题）
- **崩溃 / 卡死**：去 [Issues](https://github.com/Qc-riviere/EchoMind-public/issues/new) 提报，附操作系统 + 版本号 + 触发步骤
